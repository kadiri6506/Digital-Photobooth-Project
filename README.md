# 📸 Digital Photobooth Engine | Dijital Fotoğraf Kabini Motoru

## 🇹🇷 Proje Hakkında
Bu proje, **Ankara Üniversitesi Yazılım Mühendisliği** öğrencisi tarafından geliştirilmiş, **DIGIFEST'26** etkinliği için hazırlanan profesyonel bir dijital fotoğraf kabini motorudur. Görüntü işleme teknolojilerini kullanarak katılımcılara interaktif ve yüksek kaliteli bir hatıra deneyimi sunar.

### ✨ Öne Çıkan Özellikler
* **Gerçek Zamanlı Önizleme:** Kullanıcı dostu arayüz ile anlık kamera akışı.
* **Akıllı Maskeleme:** Fotoğrafların çerçevelere kusursuz sığması için Gaussian Blur tabanlı kenar yumuşatma algoritmaları.
* **Otomatik Kolaj:** Çekilen karelerin Polaroid veya özel şerit formatlarında anında birleştirilmesi.
* **Özel Tasarım Desteği:** Profesyonel çerçeve şablonlarıyla tam uyum.

## 🔌 Donanım Gereksinimleri
Bu proje, yüksek kaliteli görüntü işleme sonuçları için harici kamera desteği ile optimize edilmiştir.
* **Kamera:** Harici Web Kamerası (Testlerde **Iriun Webcam** kullanılmıştır).
* **Bağlantı:** Stabil bir görüntü akışı için USB veya yüksek hızlı yerel ağ bağlantısı önerilir.

## 🇺🇸 About the Project
This is a professional digital photobooth engine developed by a **Software Engineering** student at **Ankara University**, specifically prepared for the **DIGIFEST'26** event. Utilizing advanced image processing techniques, it provides an interactive and high-quality souvenir experience for participants.

### ✨ Key Features
* **Real-time Preview:** Instant camera stream with a user-friendly interface.
* **Smart Masking:** Gaussian Blur-based anti-aliasing algorithms for seamless photo-to-frame fitting.
* **Automatic Collage:** Instant merging of captured shots into Polaroid or custom strip formats.
* **Custom Template Support:** Full compatibility with professional frame templates.

## 🔌 Hardware Requirements
This project is optimized with external camera support for high-quality image processing results.
* **Camera:** External Webcam (**Iriun Webcam** was used in tests).
* **Connection:** USB or high-speed local network connection is recommended for stable image distribution.
